# quick-bug-reporter-react-native

Shake-to-report bug reporter for React Native apps — screenshot capture, video recording, annotation, and Linear/Jira integration.

> **Status:** This package is under active development. See [REACT_NATIVE_SDK.md](../../REACT_NATIVE_SDK.md) for the full implementation plan.

## Planned Features

- **Shake to report** — Trigger bug reporting by shaking the device
- **Screenshot capture** — Capture current screen via `react-native-view-shot`
- **Video recording** — Screen recording via `react-native-nitro-screen-recorder`
- **Annotation** — Draw-to-highlight using `@shopify/react-native-skia`
- **Network logging** — Automatic fetch interception (shared with web SDK)
- **Console capture** — Console log and JS error capture (shared with web SDK)
- **Integrations** — Linear and Jira (shared with web SDK)
- **Bottom sheet UI** — Native-feeling report form via `@gorhom/bottom-sheet`

## Requirements

- React Native 0.72+
- React 18+ or 19+
